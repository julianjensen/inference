> A JSDOC substitue that will parse doc tags as well as closure annotations.


## Install

```sh
npm i jsdoc-tag-parser
```

## Usage

```js
const 
    jsdocTagParser = require( 'jsdoc-tag-parser' );

jsdocTagParser() // true
```
